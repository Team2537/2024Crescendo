Limelight

class Limelight(table: NetworkTable) : AutoCloseable

Represents a single limelight unit (vision).

The limelight gets its data from a NetworkTable. This class allows simple access to various tables through more descriptive name and properties rather than raw NetworkTable.getEntry. It also manages its own ShuffleboardTab to display information about the limelight unit.
